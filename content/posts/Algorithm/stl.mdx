## 배열
```c++
// 정적 배열 선언 방법
char array[20] = { 'h', 'e', 'l', 'l', 'o', ',', ' ', 'w', 'o', 'r', 'l', 'd', '!', '!'};
int numbers[5] = { 1, 2, 3, 4, 5 };


// 동적 배열 선언 방법
int n; std::cin >> n;
char *d_array = new char[n];

// 배열 사용법
array[1] = 'c';
std::cout << array[2];  // 문자 출력 
std::cout << array;     // 문자열 출력
```
연속된 메모리 공간을 가지는 자료구조입니다. 일반적인 PS(Problem Solving)에 제일 자주 사용됩니다.  
인덱스 기반으로 접근을 해 빠르게 원소 참조가 가능하다는 장점이 있습니다.  
할당된 크기가 부족해 새로운 원소가 들어갈 자리가 없으면, 새롭게 메모리를 할당받고(보통 기존 크기의 2배) 원소들을 복사하여 옮기는게 보통입니다.  
잦은 메모리 할당, 복사를 줄이기 위해 미리 메모리를 일정 크기 할당해두고 사용하는게 정석입니다. 
```c++
// 배열 크기 부족시 동작
int* newArray = new int[array.size * 2];        // 새 배열 할당
std::copy(array, array + array.size, newArray); // 기존 원소 복사
delete[] array;     // 기존 배열 메모리 해제           
```
### vector 컨테이너 
[\<vector\>](https://cppreference.com/w/cpp/container/vector.html)
```c++
// std::vector<T> 기본 사용법
std::vector<int> vec;   // vector 선언
vec.reserve(n: int)     // 메모리 크기 할당
vec.push_back(n)        // 원소 삽입. 뒤로 들어감 공간이 부족하면 알아서 메모리 재할당이 이뤄짐
std::vector<int> vec1(n, a)     // 선언과 동시에 메모리 할당 + 원소 삽입. 
```
