# 사이트 설명
- 냥코대전쟁 모바일 게임의 월간미션 도우미 사이트 제작을 목적으로 만들었습니다. 
- 아군 캐릭터 조회 페이지와 적 캐릭터 조회 페이지도 간략하게 구현돼있습니다.
- 본능 해방을 통해 추가되는 아군 캐릭터 능력은 조회되지 않습니다.
- BCU 데이터를 이용하여 제작하였습니다. 

# 캐릭터 조회 페이지
![필터](/1.JPG)
- 최대한 앱 내부의 필터와 비슷하게 만들었습니다.
- 추가적으로 마녀 킬러와 사도 킬러, 역 워프 필터 기능을 추가하였습니다.
![다이얼로그](/2.JPG)
- 다이얼로그에서 캐릭터 레벨별 능력치를 확인하실 수 있습니다. 
- 보물 해금이나 스탯 상승 패시브는 적용되지 않습니다. 

- https://crusthack.github.io/catbattle/cat/000
- 위와 같은 링크로 뒤에 캐릭터 아이디를 넣어 해당 캐릭터에 대한 페이지로 이동하실 수 있습니다.
캐릭터 ID는 bcu 데이터의 캐릭터 번호 그대로 넣었으며, 냥코DB 일본 사이트에 나오는 캐릭터 ID보다 1 낮은 값입니다.

# 월간미션 도우미 페이지

![도우미 페이지](3.JPG)

- 필터에서 스테이지 종류와, 단게(1성 2성 3성 -> 1장 2장 3장에 대응), 적 이름을 선택하고 추가 버튼을 누르시면 해당 미션이 추가됩니다. 
- 추가된 미션에 따라 추천 스테이지에서 추천되는 스테이지가 나옵니다. 제일 빠르게 클리어할 수 있는 스테이지 / 통솔력이 적게 드는 스테이지 정렬 기능이 있습니다.
- 추천 스테이지 정렬 규칙은, 앞에있는 스테이지(세계편 -> 미래편 -> 우주편 -> 레전드스테이지), 앞선 단계(1성 -> 2성 -> 3성 -> 4성),  
이후 해당 맵에서 목표 적이 제일 많이 등장하는 스테이지 순서로 정렬됩니다. 
- 미션을 추가하실 때, 적을 선택하시면 추가버튼으로 포커스가 이동하고, 추가버튼을 클릭(엔터)하시게 되면 적 이름 검색창으로 포커스가 이동하게 되어,  
같은 스테이지, 같은 단계에서의 적을 여러개 추가할 때 키보드 입력만으로 편하게 추가하실 수 있습니다.
